How to Install and Run Cypress

Step 1: Prerequisites
You need Node.js (v16+) installed. Check with:
node --version
npm --version


Step 2: Install Cypress Dependencies
Open terminal and run:
arkad_mines_mern-stack/automated-tests"
npm install

This installs Cypress (defined in package.json).

Step 3: Start Your Application (All 3 Services Must Be Running)
Before running tests, you need all 3 parts of your MERN app running in separate terminal tabs/windows:

Terminal 1 - Backend (port 4000):
arkad_mines_mern-stack/ARKAD_Mines_backend
npm install
npm run dev

Terminal 2 - Buyer Frontend (port 5173):
arkad_mines_mern-stack/ARKAD_Mines front-end
npm install
npm run dev

Terminal 3 - Admin Frontend (port 5174):
arkad_mines_mern-stack/ARKAD_Mines admin
npm install
npm run dev



Step 4: Run Cypress Tests
Open a 4th terminal and navigate to the tests folder:
arkad_mines_mern-stack/automated-tests

3 ways to run tests:

Option A: Cypress GUI:
npm run cy:open

This opens a visual interface where you can:
Select "E2E Testing"
Pick a browser (Chrome recommended)
Click any test file to run it


Option B: Run All Tests in Terminal:
npm run cy:run

Runs all 10 test files in the terminal without a visible browser. Results are printed as text.

Option C: Run a Single Use Case
npm run test:001    # Browse Catalog
npm run test:002    # Track Orders
npm run test:003    # Register Block
npm run test:004    # Request Quotation
npm run test:005    # Approve/Reject Quotation
npm run test:006    # Place Order
npm run test:007    # Process Payment
npm run test:009    # Download Invoices
npm run test:010    # Manage Users
npm run test:010b   # Admin Create Quotation

Option D: Run with Visible Browser
npm run cy:run:headed
